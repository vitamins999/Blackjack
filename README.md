# Blackjack

## What is this?

A simple console implementation of the game Blackjack, written in C#.

## What the rules?

Check them out [here](https://bicyclecards.com/how-to-play/blackjack/).

## Anything else to know?

The current version doesn't support betting, but it is planned in the next update.  As such, you only have the option to stand or hit when drawing a card.

Have fun!
